# System Programming Repository

이 저장소는 시스템 프로그래밍 실습 코드 및 예제를 포함하고 있습니다.


## 📄 주요 파일
- `main` 브랜치: `README.md`, `main.c`, `func.c`
- `your-branch` 브랜치: 새로운 기능이 추가된 `new_feature.c`

## 🚀 브랜치 변경 및 확인 방법
```bash
# 브랜치 목록 확인
git branch -a

# 특정 브랜치로 이동
git checkout 브랜치이름

# 최신 브랜치 업데이트
git pull origin 브랜치이름
